The StitchPerfect philosophy

Traditional leatherwork cuts the piece first, then hopes the stitches space evenly. They rarely do — so the maker fudges one short stitch. That correction is visible.

StitchPerfect inverts the process. You set the pitch (distance between stitch holes) and the tool adjusts the piece dimensions by a fraction of a millimetre until every run lands on a whole number of stitches. That tiny adjustment is invisible to the eye.

Rule: Move the leather, never the stitch. Pitch is sacred.

Drawing tools

▷ Select
Click any object to select it. Drag to move. Shift+click = multi-select. Drag empty canvas = rubber-band select.
Line
Click to place vertices. Double-click or Enter to finish. Shift = snap to 45°. Creates an open stitch line.
Bezier
Click = sharp corner. Click+drag = smooth curve (drag sets the forward handle). Click near start or Enter to close.
Rect
Drag to draw. All 4 edges stitched by default. Click any edge to cycle: stitched → open → hidden. Drag corners to resize.
Oval
Drag bounding box to draw an ellipse. Drag N/S handles to change height; E/W for width.
Trap
Drag to create a trapezoid (wider top, 20% narrower at bottom). All 4 corners are freely draggable afterwards.
Poly
Click to place vertices. Click near the first vertex or press Enter to close. Click any edge to cycle state.
T Text
Click to place a text label. Enter content when prompted. Drag to reposition.
↔ Dim
Click two points to place a dimension line with arrows and a length label.

Stitch parameters (Stitch tab)

Pitch

Distance between adjacent stitch holes in millimetres. Matches your pricking iron. Standard values: 3 / 3.38 / 3.85 / 4 / 5 / 6 mm. Never changes to fit geometry — geometry changes to fit pitch.

Margin

Distance from the cut edge to the stitch line. Typical leatherwork uses 3–4 mm. Adjust with the slider (1–15 mm, half-mm steps).

Mark type

How each stitch position is rendered:

  • Hole ○ — open circle (1 mm diameter)
  • Dot ● — filled dot
  • / Slash — 2 mm line at +45° to the local edge direction
  • \ Slash — 2 mm line at −45° to the local edge direction

Slash marks rotate automatically to follow the curve — they always sit at the correct angle relative to the stitch line, even on curves and corners.

Edge states

Click any edge (on a Rect, Poly, or T-Pocket) to cycle through three states:

Stitched (blue)
Stitch marks placed. Cut outline offset outward by margin.
Open (gray dashed)
No stitch marks. Cut outline sits flush at the drawn edge. Leather is there; no holes punched.
Hidden (invisible)
No line drawn. The cut outline still closes the piece. Use for pocket openings or fold edges.

T-Pocket template

▶
T-Pocket template — drop, resize, and mirror (coming soon)

Click T-Pocket in the Templates section to drop a card-slot piece with the correct stitch states pre-set (top hidden, ears stitched, angled sides open, bottom seam stitched).

Five orange handles control the five dimensions. The angled sides always auto-adjust:

  • Left edge handle ↔ — changes total width (symmetric)
  • Top-centre handle ↕ — changes T-bar height (ear height)
  • Inner shoulder handle ↔ — changes T-tab width (mirrors right)
  • Bottom-left corner ↔ — changes base seam width (symmetric)
  • Bottom-centre handle ↕ — changes total height
Tip: Use Flip H (Edit section) to mirror the T-Pocket for the right-hand side of a bifold.

Selecting and moving objects

The Select tool (▷) is the default. It works on all object types at once.

  • Click a piece → select it (orange highlight box)
  • Shift+click → add to / remove from selection
  • Drag on empty canvas → rubber-band select (selects all fully enclosed pieces)
  • Drag a selected piece → move it
  • Alt+click → cycle through overlapping pieces (top → bottom → top)
  • Delete → remove selected pieces

Fusing shapes (Fuse button)

▶
Building a T-pocket from a Trap + Rect (coming soon)

To combine two shapes into one (e.g. a Rect on top of a Trap to make a T-shaped piece):

  1. Draw both shapes so they overlap at the shared edge
  2. Select both (Shift+click or rubber-band)
  3. Click ⬡ Fuse shapes in the Edit section
  4. The internal shared edge is removed; a new combined polygon is created
  5. Click individual edges of the result to set stitched / open / hidden states

Exporting and printing

Open the Print tab to configure export settings.

  • Download SVG — single-file vector export. 1 mm in the file = 1 mm on paper. Print at 100% scale.
  • Download tiled pages — splits the layout across A4/Letter/A3 pages with overlap margins and registration crosshairs. Print each page, cut on the dashed line, tape together.
  • Scale bar — every exported SVG includes a 50 mm reference bar. Measure it after printing to confirm scale is correct.
Printer calibration: Print the SVG at 100% (no "fit to page"). Measure the 50 mm scale bar with a ruler. If it's not 50 mm, adjust your printer's scaling setting.
